Automotora

Un sitio de automotora modelo, donde se utiliza una API y a través de Vue.js se logra plasmar la información. A su vez se utiliza el mismo framework para mostrar en tiempo real las opciones del formulario. Una vez seleccionadas todas las opciones se utiliza el boton para filtrar el resultado y mostrarlo. En caso contrario de que no exista un modelo o para evitar resultados mal intencionados la pagina responde que no existe ninguna coincidencia.

Buscador de películas

Se escribe el titulo de la pelicula en inglés, ya que la API que consume es de un sitio en ese idioma. Ademas de utilizar AJAX y JSON, se utiliza Bootstrap para la disposición y la estética, y jQuery para la animación. Utilizo una condicional para forzar al usuario a ingresar un título, mostrando una alerta en caso de que el mismo intente ingresar un string vacío.
